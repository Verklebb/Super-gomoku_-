## 🚀 一键启动游戏

### Mac 用户
**推荐方式：** 双击运行：`启动游戏.command`

**如果上面不工作：**
- 问一下AI需要安装哪些包

### Windows 用户
双击运行：`start-game.bat`

### 手动启动
```bash
npm install
npm run dev
```
然后打开浏览器访问：http://localhost:3000/
